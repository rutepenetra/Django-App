Trabalho Prático de Base de Dados 2

No âmbito da disciplina de Base de Dados II, foi pedido o desenvolvimento de uma aplicação Web para a gestão da atividade de um grupo desportivo.

Esta aplicação Web faz uma listagem simplificada de todas as tabelas da base de dados, permitindo a algumas tabelas uma visualização completa dos dados da tabela, assim como uma atualização dos dados da mesma. Permite também a inserção de novos valores em todas as tabelas. No entanto apenas duas tabelas apresentam uma opção para apagar dados.
